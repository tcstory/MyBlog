python3打包

今天研究了一下 python3中打包，现在主要看的书是 head first pyhon 感觉还是不错的的

以我自己的实验为例

我在tcstory这个文件夹中创建了 tcstory.py 和setup.py 两个文件，用来打包

内容如下

tcstory.py

setup.py

现在解释一下，setup.py中的name 和 version 这两个参数的值会合并成 tag.gz 文件包的名字

如图

py_modules 填写的东西是你当前目录下的 .py 文件（拿来作为模块的）

简单讲解一下打包方法

python3 setup.py sdist

安装在本地的方法

sudo python3 setup.py install

安装完成后，模块被安装在如下目录（可能你的电脑有所不同，你看看安装过程就知道它被放在哪里了）

/usr/local/lib/python3.2/dist-packages

打开后看看一下

打开一下_-_egg-info文件，原来这个保存的是模块的信息

打完收工
