　　先说一下，我看了一下文档，我估计我用到的功能只有如下3个，我个人的需求还是很少的，没必要搞那么麻烦

　　:ColorV(cv) : show ColorV window 显示色彩管理窗口（我英文很烂，请见谅）

　　:ColorVEdit (ce) : Edit color text under cursor 修改当前光标下的颜色代码

　　:ColorVInsert (cii): Insert color with ColorV window. 在空白处插入一个颜色代码（16进制）

　　先给大家看一下色彩管理窗口

　　ps 这个是vim默认的按键名称，默认情况下是 ”\“ ，他在你的回车键 和backspace键的中间，有关内容请看vim文档 输入 :help mapleader

　　输入 \ c v （直接在命令模式下输入）
